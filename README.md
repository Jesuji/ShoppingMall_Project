# 🛒 쇼핑몰 프로젝트

## 📎 프로젝트 개요
이 프로젝트는 전공 시간 과제로 진행한 사용자가 상품을 검색하고, 장바구니에 담고, 주문할 수 있는 **간단한  온라인 쇼핑몰** 웹 애플리케이션 입니다. 

<br>

## ⚒️ 개발 환경
- **IDE**: IntelliJ IDEA Community
- **JDK**: OpenJDK 17 
- **Database**: mysql 9.0.1
- **Frontend**: Thymeleaf, HTML, CSS, JavaScript

<br>

## ⚒️ 기술 스택
### 📌 Backend (백엔드)
- **Java 17** 
- **Spring Boot 3.3.5**
- **JPA (Hibernate)**
- **Lombok**  

### 📌 Frontend (프론트엔드)
- **Thymeleaf** 
- **HTML / CSS / JavaScript**

### 📌 Database (데이터베이스)
- **MySQL 9.0.1** 
- **H2 Database**
